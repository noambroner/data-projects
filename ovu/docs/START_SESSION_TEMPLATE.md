# 🚀 תבנית לפתיחת סשן חדש

## מה להגיד לסוכן AI בתחילת כל סשן

### אופציה 1: פשוטה (מומלצת)

```
בוא נתחיל סשן חדש. המשימה שלי היא: [תאר את המשימה]
```

### אופציה 2: מפורטת (אם יש הקשר מיוחד)

```
בוא נתחיל סשן חדש.

המשימה: [תאר את המשימה]
הקשר: [אם יש משהו חשוב לדעת מהסשן הקודם או בעיות ידועות]
עדיפות: [גבוהה/בינונית/נמוכה]
```

## מה הסוכן יעשה אוטומטית

לאחר שתגיד את זה, הסוכן:

1. ✅ **יקרא את כל הקבצים הנדרשים:**

   - README.md
   - DevelopmentGuidelines.tsx (כל 17 ההנחיות)
   - SESSION_HANDOFF.md (הסשן הקודם)

2. ✅ **יציג לך סיכום:**

   - כמה sessions יש
   - מה היה בסשן האחרון
   - מספר הסשן הבא (#8, #9, וכו')
   - הנחיות קריטיות לזכור

3. ✅ **יתחיל לעבוד על המשימה שלך**

## דוגמאות

### דוגמה 1: משימה פשוטה

```
בוא נתחיל סשן חדש. המשימה שלי היא להוסיף כפתור "שמור" בדף ההגדרות.
```

### דוגמה 2: משימה עם הקשר

```
בוא נתחיל סשן חדש.

המשימה: לתקן את ה-bug ב-preferences endpoint שמחזיר שגיאה 500
הקשר: המשתמשים מתלוננים שהם לא יכולים לשמור העדפות
עדיפות: גבוהה
```

### דוגמה 3: משימה מורכבת

```
בוא נתחיל סשן חדש.

המשימה: להוסיף תמיכה ב-dark mode לכל המערכת
הקשר: יש כבר CSS variables מוגדרות, צריך רק להשתמש בהן
עדיפות: בינונית
```
