# 🚀 התחלה מהירה - מחשבון מס מכירה

## צעדים מהירים להפעלת האפליקציה:

### 1️⃣ קבל API Key
1. גש ל-https://api-ninjas.com
2. לחץ על "Sign Up" והירשם (חינם!)
3. לאחר ההרשמה, העתק את ה-API Key שלך

### 2️⃣ התקן תלויות

```bash
# התקן Python packages
pip install -r requirements.txt
```

### 3️⃣ הגדר API Key

צור קובץ `.env` בתיקיית הפרויקט והוסף:

```
API_NINJAS_KEY=your_api_key_here
```

**החלף את `your_api_key_here` ב-API Key האמיתי שלך!**

### 4️⃣ הפעל את השרת

```bash
python app.py
```

### 5️⃣ פתח בדפדפן

```
http://localhost:5000
```

## 🎉 זהו! המערכת פועלת!
