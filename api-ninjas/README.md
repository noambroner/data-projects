# 🛒 מחשבון מס מכירה - ארצות הברית

אפליקציית ווב פשוטה ויפה לחישוב מס מכירה (Sales Tax) בארצות הברית, מבוססת על API של [API Ninjas](https://api-ninjas.com).

## ✨ תכונות

- 📍 חישוב מס מכירה לכל מיקוד (ZIP Code) בארה"ב
- 💵 תמיכה בכל סכום רכישה
- 📊 פירוט מלא של כל סוגי המסים:
  - מס מדינתי (State Tax)
  - מס מחוז (County Tax)
  - מס עירוני (City Tax)
  - שיעור מס כולל
- 🎨 ממשק משתמש מודרני ונוח בעברית עם תמיכה מלאה ב-RTL
- 📱 עיצוב רספונסיבי - עובד מצוין על כל המכשירים
- 🔑 **ממשק הגדרת API Key** - הזנת מפתח API ישירות דרך הממשק!

## 📋 דרישות מקדימות

- Python 3.8 ומעלה
- חשבון ב-[API Ninjas](https://api-ninjas.com) (חינם!)

## 🚀 התקנה והפעלה

### 1. שכפול הפרויקט

אם הפרויקט נמצא ב-Git:
```bash
git clone <repository-url>
cd api-ninjas
```

### 2. יצירת סביבה וירטואלית והתקנת תלויות

```bash
# יצירת סביבה וירטואלית
python3 -m venv venv

# הפעלת הסביבה הוירטואלית
# Linux/Mac:
source venv/bin/activate
# Windows:
# venv\Scripts\activate

# התקנת תלויות
pip install -r requirements.txt
```

### 3. הגדרת API Key

**🔑 פשוט ונוח!** האפליקציה כוללת ממשק להזנת API Key ישירות:

1. הירשם ל-[API Ninjas](https://api-ninjas.com) (חינם!)
2. העתק את ה-API Key שתקבל
3. הפעל את השרת (`python app.py`)
4. גש ל-`http://localhost:5000/api-key-setup`
5. הדבק את המפתח ולחץ "שמור"

**זהו!** המפתח נשמר באפליקציה והמערכת מוכנה לשימוש.

### 4. הפעלת השרת

```bash
python app.py
```

השרת יעלה על הכתובת: `http://localhost:5000`

## 📖 שימוש

1. פתח את הדפדפן וגש לכתובת `http://localhost:5000`
2. מלא את הפרטים:
   - **מיקוד (ZIP Code)** - הזן מיקוד בן 5 ספרות מארצות הברית
   - **סכום הרכישה** - הכנס את סכום הרכישה בדולרים (לפני מס)
3. לחץ על "חשב מס מכירה"
4. קבל פירוט מלא של כל המסים!

## 📊 מה האפליקציה מחשבת?

האפליקציה מחזירה פירוט מלא הכולל:

- **פרטי המיקום:**
  - עיר ומדינה
  - מיקוד
  - סכום הרכישה המקורי

- **שיעורי המס:**
  - שיעור מס כולל
  - שיעור מס מדינתי
  - שיעור מס מחוז
  - שיעור מס עירוני

- **סכומי המס:**
  - סך המס
  - מס מדינתי
  - מס מחוז
  - מס עירוני

- **סיכום:**
  - סכום כולל לתשלום (כולל מס)
